## Installation

You can install the package via composer:

``` bash
composer require jjpmann/real-magnet-sdk
```

## Usage

### Current Methods

* addRecipient
* editRecipient
* searchRecipients
* getRecipientGroups
* editRecipientGroups
* getRecipientFields
* getGroups
* addGroup
* getGroupDetails
* getGroupCategories
* getMessageCategories
* subscribeRecipient
* getSubscribers



---
Official RealMagnet API documentation can be found [here](https://dna.magnetmail.net/ApiAdapter/Rest/help
).
